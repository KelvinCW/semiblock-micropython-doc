# List

A **list** stores several values in order, like `[20, 50, 30]`. You can add to a
list, remove from it, sort it, and read items out by position. The List category
has blocks for all of this.

Most list blocks are **statement blocks** that act on a list you have named.

## What you will learn

- [Creating, appending, inserting, removing](basics.md)
- [`pop`, `sort`, `reverse`, `len`](manipulate.md)
- [Indexing and slicing](slicing.md)

## A quick taste

```python
scores = [20, 50, 30]
scores.append(40)
scores.sort()
print(scores)
```
